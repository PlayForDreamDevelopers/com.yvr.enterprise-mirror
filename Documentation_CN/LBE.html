大空间(LBE) 支持

Note

可参考我们的开源项目 LBE 示例 获取更多信息

为使用 LBE 相关的接口，你首先需要勾选 YVRManager 脚本上的 Large Space Support 选项：

LBE 相关的接口都定义在 LBEPlugin 中，目前主要包含有一下接口：

接口 说明
GetGroundDistance 获取地面距离
SetMarkerDetectionEnable 设置标记检测是否启用
SetMarkerTrackingUpdateCallback 设置标记跟踪更新回调，回调在识别到标记时被调用，参数中包含 MarkerTrackingUpdateData 结构体
SetRecenterEnable 设置重新定位是否启用
GetRecenterEnable 获取重新定位是否启用状态

注册 Marker 触发回调的示例为：

// ...
LBEPlugin.instance.SetMarkerDetectionEnable(true);
LBEPlugin.instance.SetMarkerTrackingUpdateCallback(OnReceiveMarkerTrackingUpdateData);

private void OnReceiveMarkerTrackingUpdateData(MarkerTrackingUpdateData data)
{
    // ...
}
