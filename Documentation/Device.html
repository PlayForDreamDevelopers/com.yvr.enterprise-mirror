Device Information and Control

Note

Refer to our open‐source project Device Sample for more details.

All device-related information and controls can be implemented via [DeviceMgr]. The following interfaces are supported. Items marked with * are writable (settable):

Information Description
btMac Bluetooth MAC address
wifiMac Wi‑Fi MAC address
deviceModel Device model
deviceSn Device serial number
osVersion Operating system version
volume * Current volume
maxVolume Maximum volume
volumeAdjustmentRestricted * Volume adjustment restriction status
brightness * Current brightness
brightnessAdjustmentRestricted * Brightness adjustment restriction status
